Class Curve

A cubic Bézier curve.

For details on Bézier curves, see Bézier curves on Wikipedia.

y coordinate of curve point P3
Curve.p
table of curve points. Use this to access points in a curve by index. Note that for consistency with Lua, the indices here are one-based, even though traditional Bézier terminology is 0-based. So keep in mind that P0 = p[1], P1 = p[2], P2 = p[3], P3 = p[4].

Usage:

    print(graphics.paths[1].subpaths[1].curves[1].p[1].x) -- get P0.x
graphics.paths[1].subpaths[1].curves[1].p[4].y = 7 -- set P3.y
